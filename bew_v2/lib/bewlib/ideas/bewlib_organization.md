### module loader ?


Load all modules

```lua
	bew.loadModule(bew.module.ALL)
```

Load only specified module(s)

```lua
	bew.loadModule({
		bew.module.UTILS,
		bew.module.KEYMAP,
		bew.module.TAG,
		bew.module.WORKSPACE
	})
```

Load all modules but not some module(s)

```lua
	bew.loadModule({ bew.module.DIALOG }, { invert = true })
```
